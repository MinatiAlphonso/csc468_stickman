CSS

1. The background of a page must include a color other than white or a background image.
This can be found in the following files: mainStyle.css, fileStyle.css, helpStyle.css.
I used the same background for all three pages.
You can find the style under 'body'.

2. There must be a custom CSS header included in all pages (note, not heading).
This can be found in the following files: mainStyle.css, fileStyle.css, helpStyle.css.
I used the same header for all three pages.
You can fund the style under 'header'.

3. The CSS on the navigation bar must include a background color/image rule.
This can be found in the following files: mainStyle.css, fileStyle.css, helpStyle.css.
I used the same navigation bar for all three pages, it has three options: Main, File, and Help.
You can find the style under 'nav', '.navitem', and 'a:link, a:visited'.

4. There must be a custom link ('a') rule.
This can be found in the following files: mainStyle.css, fileStyle.css, helpStyle.css.
I used it mainly for the three options - Main, File, and Help.
You can find the style under 'a:link, a:visited'. In helpStyle.css only, there is another style
found under 'a:link.help, a:visited.help' for the links to the different sections of the 'Help' page.

5. There may not be notable gaps between the stickman images.
This can be found in mainStyle.css, under 'div'.

6. The stickman area must include a border of some sort.
This can be found in mainStyle.css under '.canvas'.

7. The help page headings must have a font style change
This can be found in Help.css under 'h2'.
